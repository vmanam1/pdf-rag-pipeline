# Contributing

1. Open an issue describing a bug or proposed change before substantial work.
2. Create a focused branch from `main`.
3. Keep secrets and generated indexes out of commits.
4. Add or update tests for behavioral changes.
5. Run relevant tests and Django checks locally.
6. Open a pull request describing the motivation, implementation, and validation performed.

By contributing, you agree that your contribution is licensed under this repository's MIT
License.
